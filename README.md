# ExemploSeguros-BackEnd

- API de suporte para a aplicação de testes de um seguro em Angular 4.

# Implemented Tools
- .NET Core
- ASP.NET WebAPI
- C#
- EntityFramework 7
   - Code First
   - FluentAPI
- Dapper 1.5.2
- Azure SQL Server
- AutoMapper 6.02
- Native Injector
- Fluent Validator

# Patterns Applied
- DDD - Domain Drive Design
- SOLID
- CQRS - Command Query Responsibility Segregation
- CrossCutting
- IOC - Dependency Injection
- Self-Validation
- Event Sourcing
- Service Bus
- Factory Methods
